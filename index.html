Pixel Hammer Help & Hotkeys

Left Panel — Tools / Brush / Canvas

  • Pencil draws with the current color.
  • Line places straight pixel lines.
  • Eraser removes pixels.
  • Fill flood-fills connected pixels.
  • Select highlights, moves, and captures custom brush regions.
  • Eyedropper picks a color from the canvas.
  • Brush Size changes the active brush shape or size.
  • Mirror turns mirror drawing on or off.
  • H mirrors horizontally. This is the default axis when Mirror turns on.
  • V mirrors vertically. If H and V are both checked, both directions are used.
  • Toggle Grid shows or hides the drawing grid.
  • Onion Skin shows nearby animation frames when available.
  • Zoom changes the canvas view without changing export size.
  • Import PNG loads an image into the current frame.
  • Import New Frame loads an image into a newly created frame.
  • Import Sheet slices a spritesheet into frames.
  • Export PNG exports the current frame.
  • Sheet H / Sheet V / Wrap Sheet export animation frames as spritesheets.
  • Canvas Size changes document resolution.

Right Panel — Edit / Workspace / Color

  • Undo / Redo step backward or forward through history.
  • Frames / Tileset Preview switches the workspace view.
  • ADD Frame adds a frame quickly.
  • Hide Workspace Panel collapses the workspace section.
  • Prev / Next move between frames when frame arrows are shown.
  • Add / Dup / Delete Frame manage animation frames.
  • Frame Hold changes how long the selected frame stays on screen.
  • Play / Stop preview animation playback.
  • FPS changes playback speed.
  • Mode switches playback behavior such as Loop or Ping-Pong.
  • Color picker / shade bar / palettes control color choice and saved swatches.
  • Variant controls generate and reroll color variants.
  • Layer controls appear when multi-layer mode is enabled.
  • Preview panel controls update based on frame count, layer mode, and workspace mode.

Hotkeys

  • Ctrl + Z Undo
  • Ctrl + Y Redo
  • Ctrl + Shift + Z Redo
  • Ctrl + S Save project file
  • Ctrl + O Load project file
  • P Pencil
  • N Line tool
  • R Rectangle tool
  • C Ellipse tool
  • I Eyedropper
  • O Onion Skin on/off when available
  • V Generate variant
  • B Reroll variant
  • M Toggle Mirror on/off
  • Arrow Left / Arrow Right Frame navigation or frame movement depending on frame mode
  • Arrow keys nudge the active selection when a floating selection exists
  • [ move timeline window left
  • ] move timeline window right
  • Space Play / Stop playback
  • Enter confirm line, selection, or active shape action
  • Escape cancel or clear the active line / shape / selection state
  • F1 open or close this help sheet
  • ? open or close this help sheet
